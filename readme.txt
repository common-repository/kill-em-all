== Description ==

It's a very simple and useful plugin!
Spam wiper with one click only from admin menu and all spam gone!
You dont have to do more than one click to clean all spam.

== Installation ==

1. Upload 'killemallDV01.php' to the /wp-content/plugins/killemall/ directory.
 	
2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Click 'Kill Em All' menu from the left and your spam will be wiped out!
